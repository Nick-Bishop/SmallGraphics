# SmallGraphics
This code is intended to be a small graphics library for the LaFortuna to make it easier to draw onto the screen.

Features:
- Bresenham Lines
- Bresenham Circles
- Color shading
- Bezier curves
- Anti-aliased lines

Features missing:
- Ellipse drawing
- B-Splines
- Shape filling (flood fill)

Code used: \n
